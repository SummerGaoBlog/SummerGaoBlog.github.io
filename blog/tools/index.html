# 常用工具软件

# WireShark

WireShark是广大黑客、网络工程师的最爱，极大的方便我们抓取、分析网络数据包。

# VMware

VMware虚拟机，如果硬件资源有限，如果硬件资源有限，那么使用一台电脑当N台电脑使用。

# MarkdownPad

MarkdownPad是Windows上最好用的Markdown编辑器，也是广大博客的最爱。

# Mindmanager

思维导图，据说是每个产品经理必备的软件。使用思维导图跟踪调节流程，绘制软件架构将会非常方便。

# Fiddler

Fiddler是一个http协议调试代理工具，能够非常方便的查看HTTP通信数据。
